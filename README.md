# Flappy Plane
## Descrição
* Um jogo de um avião que cai para baixo e fica pulando e tem que desviar de obstaculos
## Créditos
* Ideia do curso de desenvolvimento de jogos da [Noneclass](https://noneclass.com.br)
* Assets do [Kenney](https://kenney.nl)
* Feito na Unity
